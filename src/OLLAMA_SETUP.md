# Ollama Setup

# دليل تثبيت المساعد الذكي المحلي
## وزارة الداخلية الأردنية — مديرية التنمية المحلية

---

## الخطوة 1: تثبيت Ollama على Windows Server

1. افتح المتصفح وانتقل إلى: **https://ollama.com/download/windows**
2. حمّل ملف التثبيت `OllamaSetup.exe`
3. شغّل الملف وأكمل التثبيت (يحتاج صلاحيات مسؤول)
4. بعد التثبيت، افتح CMD وتحقق:
   ```
   ollama --version
   ```

---

## الخطوة 2: تحميل النموذج

افتح CMD وشغّل أحد الأوامر التالية:

**للأجهزة بـ 16GB RAM أو أكثر (موصى به):**
```
ollama pull llama3.1:8b
```

**للأجهزة بـ 8GB RAM:**
```
ollama pull phi3:mini
```

> التحميل يحتاج إنترنت مرة واحدة فقط (4-5 GB). بعدها يعمل offline.

---

## الخطوة 3: تشغيل Ollama كخدمة

```
ollama serve
```

للتشغيل التلقائي مع Windows، افتح Task Scheduler وأضف مهمة تشغيل `ollama serve` عند بدء النظام.

---

## الخطوة 4: تشغيل خادم المساعد

في مجلد المشروع، افتح CMD وشغّل:
```
python src/ai_server.py
```

للتشغيل التلقائي، أضف هذا الأمر أيضاً في Task Scheduler.

---

## الخطوة 5: تغيير النموذج (اختياري)

افتح `src/ai_server.py` وغيّر السطر:
```python
MODEL = "llama3.1:8b"   # غيّر هنا
```

---

## التحقق من عمل النظام

افتح المتصفح وانتقل إلى:
```
http://localhost:5000/health
```

يجب أن يظهر:
```json
{"status": "ok", "model": "llama3.1:8b", "ollama": "running"}
```

---

## ملاحظة للمستخدمين في المحافظات

إذا كان النظام على سيرفر الوزارة، غيّر في `src/inject_ai.py`:
```python
AI_SERVER = 'http://localhost:5000'
# إلى:
AI_SERVER = 'http://[عنوان-سيرفر-الوزارة]:5000'
```
ثم أعد بناء الـ Dashboard.
