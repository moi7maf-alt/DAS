# نظام التخطيط التنموي الاستراتيجي
## دليل التثبيت السريع لفني الوزارة

### المتطلبات
- Windows Server (أو Windows 10/11)
- Python 3.11+ (حمل من https://python.org — تأكد من تفعيل "Add to PATH")

### طريقة التثبيت (دقيقتان)

**الخطوة 1:** انقر مرتين على `install_service.bat`

**الخطوة 2:** ستظهر نافذة تطلب "تشغيل كمسؤول" → اختر **نعم**

**الخطوة 3:** ستظهر نافخة سوداء تكتب فيها رسالة نجاح → اضغط أي مفتاح

**الخطوة 4:** الخادم يشتغل تلقائياً عند كل إقلاع للويندوز.

للتشغيل فوراً بدون إعادة تشغيل: انقر مرتين على `start_server.bat`

### الدخول للموقع
- المتصفح: http://localhost:5000
- **حساب المدير:** admin / admin123
- **حسابات المحافظات:** English username (مثل `karak.gov`) أو Arabic (مثل `الكرك`)، كلمة السر `a1b2c3`

### ⚠️ قبل الإطلاق الرسمي
1. غيّر كلمة سر `admin` في ملف `ai_server.py` (ابحث عن `admin123`)
2. غيّر كلمات سر المحافظات من `a1b2c3` إلى كلمات أقوى

### الملفات في هذا المجلد
| الملف | الوظيفة |
|-------|---------|
| `ai_server.py` | ملف الخادم الرئيسي (القلب) |
| `dist/index.html` | لوحة المحافظات |
| `dist/landing.html` | الصفحة الرئيسية |
| `dist/login.html` | صفحة تسجيل الدخول |
| `dist/audit.html` | سجل التدقيق (للمدير فقط) |
| `install_service.bat` | تثبيت الخدمة (شغله مرة واحدة كمسؤول) |
| `uninstall_service.bat` | إلغاء تثبيت الخدمة |
| `start_server.bat` | تشغيل الخادم يدوياً (للتجربة) |
| `start_daemon.vbs` | يشغل الخادم بصمت (يستخدمه Windows Task Scheduler) |
