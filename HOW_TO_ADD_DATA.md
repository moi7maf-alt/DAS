# دليل إضافة جداول بيانات جديدة

## أولاً: جداول `Data/input/` — بيانات المقطع العرضي

هذه الجداول تُغذّي درجات القطاعات الـ 8 في تبويب التحليل التنموي.

### متطلبات الجدول

| المتطلب | التفاصيل |
|---------|----------|
| عمود المحافظة | يجب أن يحتوي على أسماء المحافظات الـ 12 |
| أسماء المحافظات | العاصمة، البلقاء، الزرقاء، مأدبا، إربد، المفرق، جرش، عجلون، الكرك، الطفيلة، معان، العقبة |
| الصيغة | `.xlsx` فقط |
| الخلايا الفارغة | اتركها فارغة أو ضع 0 — لا تضع شرطة (-) |

### الخطوات

**الخطوة 1:** ضع الملف في `Data/input/`

**الخطوة 2:** افتح `src/generate_data.py` وأضف دالة قراءة:

```python
def load_my_new_data():
    df = read_excel("MyNewFile.xlsx")          # اسم الملف
    df = attach_canonical(df, "المحافظة")      # اسم عمود المحافظة في الملف
    df = df[df["canonical"].notna()].copy()
    df = df[df["canonical"].isin(GOVERNORATES)].copy()
    for col in df.columns:
        if col not in ("المحافظة", "canonical"):
            df[col] = df[col].apply(safe_float)
    return df.set_index("canonical")
```

**الخطوة 3:** أضف استدعاء الدالة في `main()`:

```python
my_data = load_my_new_data()
```

**الخطوة 4:** أضف حساب الدرجة (مثال — إضافة مؤشر لقطاع الصحة):

```python
def compute_health_scores(health_df, my_data_df):
    df = health_df.reindex(GOVERNORATES).fillna(0)
    my_df = my_data_df.reindex(GOVERNORATES).fillna(0)

    beds_per_10k   = normalize_scores(df["الأسرّة في المستشفيات  لكل 10,000 مواطن"])
    health_centres = normalize_scores(df["المراكز الصحية"])
    my_new_metric  = normalize_scores(my_df["اسم_العمود_الجديد"])  # ← أضف هنا

    # عدّل الوزن حسب أهمية المؤشر
    composite = (beds_per_10k + health_centres + my_new_metric) / 3
    return composite
```

**الخطوة 5:** أضف المؤشر الخام في قسم `raw_data` داخل حلقة `for gov in GOVERNORATES`:

```python
if gov in my_data.index:
    raw_data["my_new_indicator"] = safe_float(my_data.at[gov, "اسم_العمود"])
```

**الخطوة 6:** شغّل البناء:

```
python src/run_all.py
```

## ثانياً: جداول `Data/trend/` — بيانات الاتجاه الزمني

هذه الجداول تُغذّي التحليل التنبؤي (مخطط الاتجاهات والتوقعات حتى 2028).

### متطلبات الجدول

| المتطلب | التفاصيل |
|---------|----------|
| عمود المحافظة | الاسم الأول في الجدول |
| أعمدة السنوات | 2021، 2022، 2023، 2024 (أو أي نطاق زمني) |
| الصيغة | `.xlsx` فقط |

### مثال على هيكل الجدول المطلوب

| المحافظة | 2021 | 2022 | 2023 | 2024 |
|----------|------|------|------|------|
| العاصمة  | 45.2 | 46.1 | 47.3 | 48.0 |
| البلقاء  | 32.1 | 33.4 | 34.2 | 35.0 |
| ...      | ...  | ...  | ...  | ...  |

### الخطوات

**الخطوة 1:** ضع الملف في `Data/trend/`

**الخطوة 2:** افتح `src/build_predictive2.py` وأضف قراءة الملف بعد قسم "LOAD ALL FILES":

```python
# --- MY NEW INDICATOR (2021-2024) ---
my_new_indicator = {}
try:
    df = pd.read_excel(f'{TREND_PATH}/MyNewTrendFile.xlsx')
    df['_gov'] = df['المحافظة'].apply(norm)
    df = df[df['_gov'].isin(GOVS)]
    # إذا كانت الأعمدة مسماة بالسنوات مباشرة:
    for _, row in df.iterrows():
        gov = row['_gov']
        my_new_indicator[gov] = {
            yr: safe(row.get(str(yr))) for yr in [2021, 2022, 2023, 2024]
        }
    print(f'  My New Indicator: {len(my_new_indicator)} govs')
except Exception as e:
    print(f'  ERROR MyNewTrendFile: {e}')
```

**الخطوة 3:** أضف المؤشر في قاموس `INDICATORS`:

```python
'مؤشري_الجديد': {
    'label':       'اسم المؤشر للعرض',   # يظهر في الواجهة
    'unit':        'وحدة القياس',         # مثال: '%' أو 'مركز' أو 'طن'
    'lower_better': False,                # True إذا كان الأقل أفضل (مثل البطالة)
    'sector':      'اسم القطاع',          # أحد القطاعات الـ 8
    'weight':      1.0,                   # وزن المؤشر في حساب المخاطرة (0.5 - 1.5)
    'data':        my_new_indicator,      # المتغير الذي قرأت فيه البيانات
    'years':       [2021, 2022, 2023, 2024]
},
```

**الخطوة 4:** شغّل البناء:

```
python src/run_all.py
```
